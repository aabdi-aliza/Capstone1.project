# Capstone1.project
In this project,  to use NumPy, Matplotlib, and Data Analysis to analyze the NHANES dataset and present the work in a Jupyter Notebook in the form of a report.


In this project, you need to:

* Download the male and female body measurement data.
* Read the data into NumPy matrices.
* Create graphs such as histograms, boxplots, and scatterplots.
* Perform some statistical calculations.
* Write explanations and analysis in a Jupyter Notebook.

In simple terms to create a Data Analysis Report using the dataset.
